(static) formatMetadata(metadata) → {Object}

Format DICOM metadata structured according to the DICOM JSON model. Transforms the DICOM JSON representation into a more human friendly representation, where values of data elements can be directly accessed via their keyword (e.g., "SOPInstanceUID"). Bulkdata elements will be extracted and returned as a separate mapping.
