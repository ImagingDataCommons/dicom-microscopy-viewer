dicom-microscopy-viewer Examples

These examples are meant to demonstrate various aspects of the dicom-microscopy-viewer library in an interactive way. They are written such that they can be opened directly in your web browser from a local copy of the repository.

  • Rendering a single whole-slide image - The basic usage of the library to view a whole slide image
  • Handling viewer events - Use custom viewer events to handle actions
  • Simple whole-slide image viewer - List and navigate available studies/series
